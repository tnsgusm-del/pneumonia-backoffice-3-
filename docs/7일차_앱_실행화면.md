# 7일차 과제: 백오피스 프론트엔드 앱 실행 화면 및 기능 검증서

본 문서는 AI Agent를 활용하여 `app/static` 프론트엔드 템플릿 코드를 자체 구현한 RESTful API 엔드포인트와 연결하고, 기능 테스트가 정상적으로 완료되었음을 증명하는 실행 화면 기록서입니다.

---

## 🖥️ 1. 시스템 실행 환경 및 접속 주소
* **백엔드 구동 커맨드**: `fastapi run app/main.py`
* **프론트엔드 접속 URL**: `http://localhost:8000/` 또는 `http://127.0.0.1:8000/`

---

## 📸 2. 엔드포인트별 기능 동작 화면

### 🔐 REQ-USER-001 ~ 002. 회원가입 및 로그인 / JWT 보안 제어
* **기능 설명**: 비회원 상태에서 회원가입 청구를 진행하고, 가입 승인된 계정으로 로그인하여 Bearer Access Token을 정상 발급받는 화면입니다.
* **화면 캡처**:

---

### 🏥 REQ-PTNT-001 ~ 005. 환자 관리 인적 정보 및 목록 가동
* **기능 설명**: `STAFF` 권한을 가진 의료진이 신규 환자를 등록하고, 필터 조건(성별, 나이)에 따라 환자 리스트가 화면에 실시간 브라우징되는 모습입니다.
* **화면 캡처**:

---

### 🔬 REQ-PRED-001 ~ 002. 실시간 AI 폐렴 예측 추론 및 결과 조회
* **기능 설명**: 특정 진료기록 상세 페이지에서 [AI 예측 결과보기] 버튼을 트리거하여 PyTorch `SimpleCNN` 싱글톤 모델의 연산 결과(Confidence %, 판정 여부)를 화면에 즉각 렌더링하는 최종 검증 화면입니다.
