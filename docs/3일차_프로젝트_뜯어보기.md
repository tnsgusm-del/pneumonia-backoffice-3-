1. 각 디렉터리의 역할
app/core/: 애플리케이션의 핵심 설정(환경 변수, 보안 정책 등)이 모여있는 곳입니다.

app/core/db/: 데이터베이스 커넥션 풀을 생성하고 세션을 주입하는 코드가 작성되는 곳입니다.

app/models/: 데이터베이스의 테이블 구조를 정의하는 공간입니다. (SQLAlchemy ORM 모델)

app/schemas/: Pydantic을 활용해 API의 입력(Request)과 출력(Response) 데이터 포맷을 정의하고 검증하는 곳입니다.

app/repositories/: DB에 직접 접근하여 생성, 조회, 수정, 삭제(CRUD) 쿼리를 수행하는 곳입니다.

app/services/: 핵심 비즈니스 로직(AI 모델에 데이터 전달, 판독 결과 가공 등)을 수행하는 레이어입니다.

app/apis/: 클라이언트의 요청을 받는 API 엔드포인트(Router)들을 모아두는 곳입니다.

2. 주요 파일의 역할
app/main.py: FastAPI 서버의 시작점이며, 각종 미들웨어 및 라우터들을 하나로 묶어 등록합니다.

app/core/config.py: .env 파일의 환경 변수를 Pydantic Settings를 이용해 안전하게 관리합니다.

pyproject.toml & uv.lock: 프로젝트 메타데이터와 패키지 매니저(uv)가 사용할 의존성 패키지 목록 및 버전을 잠가 동기화를 보장합니다.
